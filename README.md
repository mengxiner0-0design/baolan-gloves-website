# GloveSource One - Industrial Blue Website

This version is a completely different design direction from the previous beauty-style website.

## Design Direction

- Industrial B2B procurement style
- Full-screen banner design
- Hard rectangular modules
- Horizontal product rows
- Dense comparison tables
- Procurement-focused copy
- Less rounded card / beauty / glassmorphism style

## Products Included

- PE Gloves
- Bag Pack
- Box Pack
- Individually Packed
- Rounded-Corner Pack
- Long-Strip Pack
- Nitrile Gloves
- TPE Gloves
- CPE Gloves
