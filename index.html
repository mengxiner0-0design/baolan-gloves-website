FAQ

Questions before procurement.

Does this site cover all requested products?

Yes. It includes PE gloves, bag pack, box pack, individually packed, rounded-corner pack, long-strip pack, nitrile gloves, TPE gloves and CPE gloves.

Is this version visually different from the beauty website?

Yes. This version uses full-bleed banners, hard industrial panels, horizontal range rows, dense comparison tables and a B2B procurement structure instead of soft beauty-style cards.

Can the images be replaced later?

Yes. All images are stored locally in the assets/products folder and can be replaced with real product photos.

Can it run on GitHub Pages?

Yes. It is pure HTML, CSS and vanilla JavaScript.
